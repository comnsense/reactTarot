## WEB наръчник за Таро карти - вашето дигитално ръководство в света на Таро.


## 📖 За проекта
Tarot101 е модерно уеб приложение, създадено да помогне както на начинаещи, така и на напреднали ентусиасти да разберат значението на всяка Таро карта и техните комбинации. Проектът представлява цялостна дигитална енциклопедия за всички 78 карти от класическата колода Таро.

## 🎯 Основни цели
- Образователна - Достъпна информация за всяка Таро карта
- Практическа - Помощ при тълкуване на карти и комбинации
- Интерактивна - Избор на карти и получаване на тълкувания
- Визуална - Красив и интуитивен начин за представяне на картите

---

### ✨ Функционалности
- Библиотека с карти (/)
- Разглеждане на всички 78 карти в подреден грид
- Филтриране по тип 
- Детайлна информация за всяка карта:
- Ключови думи
- Символика
- Съвет от картата

### 🤝 Помощ при тълкуване (/mycards)
 - Избор на до 10 карти от колодата
 - Автоматично откриване на комбинации: 

### 📖 Енциклопедия на комбинациите (/readings)
Пълен списък с 80+ предварително дефинирани комбинации

 - Филтриране по категории  
 - Филтриране по бои  
 - Мистичен съвет

### 🚀 Стартиране на проекта
```bash
# Клониране на репозиторито
git clone https://github.com/comnsense/reactTarot.git

# Влизане в директорията
cd reactTarot

# Инсталиране на зависимостите
npm install

# Стартиране на development сървър
npm start
Приложението ще се отвори на http://localhost:3000
 
bash
npm run build
Това създава оптимизирана версия в папка build.
```
